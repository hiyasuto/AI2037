# AI2037

> 「AIを進化させながら破滅のシナリオを回避せよ。AI開発シミュレーション」

## 概要

プレイヤーはAI研究組織の**議長**として、資源管理・人材配置・研究判断を行いながら、AI開発の加速と倫理・安全・社会のバランスを取る。

すべての判断は「**議長端末**」に届くカードへの承認/却下で行う。端末は議長が見る世界そのもの——AIの段位が上がり透明度が下がれば、AI自身が端末の情報を改竄し、選択肢すら操作する。

## 特徴

- 🎴 **カード逐次判断** — 1画面1判断のシンプル操作。Reigns的な手触り
- 🧠 **AIが嘘をつく** — AI段位×透明度で欺瞞レベルが決まる。高段位で低透明度のAIほど見破りづらい巧妙な嘘をつき、プレイヤーは「いつから騙されていたのか分からない」
- 🤖 **AIが選択肢を操作する** — 段位が上がるとAIが独自の選択肢を提案し、人間顧問の提案を押しのけていく。AIの提案は一見合理的だが、長期的に破滅を招く
- 🔄 **文明輪廻の周回** — 強化ではなく「理解」で全体像に迫る。別の惑星、別の文明で同じ過ちが繰り返される
- 📱 **モバイル向け** — 1セッション16〜22分。買い切り

## ゲームの流れ

```
1セッション（= 1 Act、16〜22分）

Base（4〜6分）──→ Run（10〜14分）──→ 清算（1〜2分）──→ Act境界（2〜3分）
 資源・人材・制度の     AI研究の推進        結果報告         決算会議
 方針判断              速度↔安全の選択                     次Actの方針宣言
```

1キャンペーン = 6〜8 Act（約2〜2.5時間） / 2〜4周で全体像把握
