# AI2037

「善意の組織がなぜ破滅するかを体験する、AI開発シミュレーション」

## 概要

プレイヤーはAI研究組織の議長として、資源管理・人材配置・研究判断を行いながら、AI開発の加速と倫理・安全・社会のバランスを取る。すべての判断は「議長端末」に届くカードへの承認/却下で行う。

- **プラットフォーム：** モバイル（買い切り）
- **1セッション：** 16〜22分
- **1キャンペーン：** 6〜8 Act（約2〜2.5時間）
- **周回：** 2〜4周で全体像が見える設計
