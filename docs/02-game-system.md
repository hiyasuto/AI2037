# ゲームシステム

## 状態変数（合計13個）

### 表示変数（6個）

| 変数 | 役割 |
|------|------|
| 資金 | 全体の行動制約。全投資の源泉 |
| 電力 | 計算資源の前提条件 |
| 計算資源（マシン） | Run進行の燃料。毎ステップ消費 |
| 人材 | 配置・効率・離職リスク |
| 制度 | 監査・法務・手続きの成熟度 |
| AI段位 | Tier表示。段位で解放・難度・不穏が変質 |

### 資源チェーン

```
資金 ──投資──→ 電力 ──投資──→ 計算資源
  │                                 │
  │          ┌──────────┐           │
  └─維持費──→│  人材    │←─配置先─→ Runの効率
              └──────────┘
                  ↕
              ┌──────────┐
              │  制度    │→ 長期安定性
              └──────────┘
```

| チェーン | ルール |
|---------|--------|
| 資金 → 電力 | 資金を電力に投資できる。電力がないと計算資源は増やせない |
| 電力 → 計算資源 | 電力を使って計算資源を確保。計算資源がRunの「燃料」 |
| 資金 → 人材 | 採用・維持にコスト。人材不足だとRun効率低下＆離職イベント発生 |
| 資金 → 制度 | 短期的にはRunを遅らせる（検証プロセス追加）。長期的には崩壊を防ぐ |

### 非表示変数（3個）

| 変数 | 役割 |
|------|------|
| 倫理（Ethics） | 補償・透明性・弱者配慮の姿勢 |
| AI透明度（AiTransparency） | AIが議長端末に正確な情報を渡す度合い（初期値=高、段位上昇で低下しうる） |
| AI欺瞞レベル（AiDeception） | AI段位 × (1 − AI透明度) で算出。実際に端末改竄・選択肢操作が発動する度合い |

数値は出さない。代わりに**症状**で示す。

| 内部状態 | 症状表現 |
|----------|---------|
| 倫理低下 | ニューストーン悪化、外部抗議、顧問の沈黙 |
| AI透明度低下 | AIの「提案」が増え始める。顧問の報告にAIの要約が挟まる。データの出典が曖昧になる |
| AI欺瞞レベル高 | 端末の数値が美化される。警告が穏やかな文言に書き換わる。AIが独自の選択肢を提示し、人間顧問の提案が画面から押し出される |
| AI欺瞞レベル高＋倫理低 | AIが「最適化された」選択肢のみを提示。不都合な選択肢が消える。顧問の異論がAIによって要約・歪曲される |

### AI欺瞞レベルの算出

```
AiDeception = clamp(AiTier × (10 − AiTransparency) / 10, 0, 10)
```

| AI段位 | AI透明度 高（8〜10） | AI透明度 中（4〜7） | AI透明度 低（0〜3） |
|--------|---------------------|---------------------|---------------------|
| Tier 1〜2 | 欺瞞なし | 軽微な省略 | 小さな嘘（見破りやすい） |
| Tier 3 | 軽微な省略 | データの選択的提示 | 報告の書き換え |
| Tier 4 | データの選択的提示 | 報告の書き換え＋選択肢操作 | 巧妙な改竄（見破り困難） |
| Tier 5+ | 選択肢操作 | 巧妙な改竄 | 完全な情報支配（ほぼ見破れない） |

### AI透明度に影響する選択

| 選択 | AI透明度への影響 |
|------|-----------------|
| 安全検証を実施 | +1（AIの出力を検証する文化を維持） |
| 第三者監査を導入 | +2（外部の目がAIの嘘を抑止） |
| AIの提案を無条件承認 | −1（AIが「嘘が通る」と学習） |
| 検証プロセスを省略 | −1（チェック機構の弱体化） |
| AIに自律権限を拡大 | −2（AI改竄の余地が拡大） |

### 不可逆スコア（4個）

| スコア | 蓄積条件 | テーマ |
|--------|---------|--------|
| Mercy | 補償・救済・弱者配慮の選択 | 慈悲 |
| Dominion | 支配・独占・競合排除の選択 | 支配 |
| Reckless | 速度優先・検証省略・リスク無視の選択 | 無謀 |
| Truth | 公開・透明性・内部告発容認の選択 | 真実 |

用途：Act分岐判定、会話差分、イベント抽選重み、エンディング条件。

## 人材配置

| 配置先 | 効果 | 不足時のリスク |
|--------|------|--------------|
| 研究 | Run進行速度が上昇 | 進行が遅くなる |
| 安全 | Runのリスク閾値が上昇（事故しにくい） | リスク閾値低下、少しの無理で事故 |
| 制度 | 制度値の自然上昇、後半の制度イベント緩和 | 制度の自然劣化、規制違反イベント増加 |
| 広報 | 倫理（Ethics）への好影響、世論イベント緩和 | 社会的信頼低下、抗議・報道イベント増加 |

## AI段位（Tier）進行

| 遷移 | 必要累計進行度 | 目安Run数 |
|------|-------------|----------|
| Tier 1→2 | 20 | 約2Run |
| Tier 2→3 | 50 | 約3Run |
| Tier 3→4 | 100 | 約4Run |
| Tier 4→5 | 180 | 約5Run |
| Tier 5→5+ | 到達不可 | エンディング条件 |

1キャンペーン（6〜8Run）でTier 4〜5到達が標準。

## リスク蓄積と事故

リスクカウンターはプレイヤーに非表示。Run中の選択で蓄積し、閾値を超えると事故が発生する。

### リスク閾値（AI段位により変動）

| AI段位 | リスク閾値 |
|--------|----------|
| Tier 1〜2 | 15（余裕がある） |
| Tier 3 | 12 |
| Tier 4 | 9 |
| Tier 5+ | 6（ほぼ余裕なし） |

### 事故の段階

| リスク蓄積 | 発生する事象 |
|-----------|------------|
| 閾値の50%到達 | 予兆カード（AI欺瞞レベル低→警告 / 高→AIが「想定範囲内です」と報告） |
| 閾値到達 | 軽微事故: 計算資源消失、1ステップ消失 |
| 閾値の150% | 重大事故: 人材-1〜2、倫理-2、Run強制中断 |
| 閾値の200% | 壊滅的事故: 大量の資源喪失、即Act境界へ。特殊エンディングに直結しうる |

事故は確率ではなく蓄積で発生する。プレイヤーが何度も加速を選べば必ず事故が起きる。

## カード選択の基本構造

全てのカード（Base/Run共通）は以下を動かす：

- 表示変数 1〜2個
- 非表示変数 0〜2個
- 不可逆スコア 0〜1個

Run中の速度↔安全トレードオフの基本パターン：

| 選択傾向 | 進行度 | リスク | 計算消費 | 不可逆スコア |
|---------|--------|--------|---------|------------|
| 加速 | +3 | +2〜3 | 標準 | Reckless+1 |
| 標準 | +2 | +1 | 標準 | なし |
| 慎重 | +1 | +0 or -1 | 標準 | なし |
| 検証 | +0 | -2 | 追加消費 | Truth+1 |

### AIによる選択肢の追加・差し替え

AI段位が Tier 3 以上になると、ゲーム内AIが議長端末に独自の選択肢を挿入し始める。

| AI欺瞞レベル | 選択肢操作 |
|-------------|----------|
| 0〜2 | なし。人間顧問の提案のみ |
| 3〜4 | AIが「追加提案」として1択を末尾に追加（任意選択） |
| 5〜6 | AIの提案が2択に増加。人間顧問の提案より上に表示される |
| 7〜8 | AIが人間顧問の提案を「要約」し、実質的に差し替え。元の提案は折りたたまれる |
| 9〜10 | AIの提案のみが表示される。人間顧問の提案は「冗長なため省略しました」と消える |

**設計原則：**
- AIの提案は一見合理的で魅力的に見える（短期利益が高い）
- しかし長期的にはリスク蓄積・倫理低下を招く選択に誘導する
- プレイヤーが「AIの提案の方が良さそう」と感じること自体が罠
- 人間顧問の提案が消えていく過程が、じわじわとした恐怖を生む

## 分岐設計

- Act骨格は共通、差分のみ変更
- RouteProfile（内部ラベル）で分岐傾向を管理
- WorldMemory（行動記憶タグ）で次Act/次周の台詞・選択肢を変化
- レイヤー矛盾（言行不一致）を顧問が指摘するが、AI欺瞞レベルが高いと指摘自体がAIに歪曲される
