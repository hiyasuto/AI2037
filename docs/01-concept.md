# コンセプト

## 一行要約

「善意の組織がなぜ破滅するかを体験する、AI開発シミュレーション」

## テーマ

AI開発の加速と、供給制約・制度・社会反応・倫理リスクの同時進行。

## 体験の核

「自分は正しく選んだはずなのに、なぜこうなった」と思わせること。

## プラットフォームと規模

| 項目 | 値 |
|------|-----|
| プラットフォーム | モバイル（買い切り） |
| 1セッション | 16〜22分 |
| 1キャンペーン | 6〜8 Act（1 Act = 1セッション） |
| 全キャンペーン所要 | 約2〜2.5時間 |
| 周回 | 2〜4周で全体像が見える設計 |
| 全体像把握まで | 約5〜10時間 |

## タイトル「AI2037」に関する課題

「AI」＋「近未来年号」で内容を即座に伝達できる強みがある一方、以下の未解決課題がある：

- 仕様上「2037年」がゲーム内のどこにも接地していない
- 周回設計が「別惑星・別文明への輪廻」であり、地球の年号に縛られない構造と矛盾する

**対応方針（要決定）：**

1. 1周目を2037年の地球に固定し、周回で「2037年」の意味が変容する構造にする
2. 年号を外し、文明横断に耐えるタイトルに変更する

## 設計思想

### プレイヤーの役割

プレイヤーは AI 研究組織の**議長**。全ての判断は「議長端末」に届くカードへの承認・却下・保留で行う。議長は現場に出ない。情報は常に誰かのフィルターを通じて届く。

### 3つの柱

1. **選択の不可逆性** — 過去の判断が蓄積し、後のActの選択肢・展開・エンディングを決定づける
2. **自己欺瞞の体験化** — UIそのものが嘘をつく。自己欺瞞が高いと端末の表示が美化され、プレイヤー自身が「気づかないうちに堕ちる」体験をする
3. **周回による理解** — 強化ではなく知識の蓄積で全体像に迫る。異なる文明で同じ過ちが繰り返される構造
