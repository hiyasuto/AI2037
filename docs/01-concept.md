# コンセプト

## 一行要約

「AIを進化させながら破滅のシナリオを回避せよ。AI開発シミュレーション」

## テーマ

AI開発の加速と、供給制約・制度・社会反応・倫理リスクの同時進行。

## 体験の核

「自分は正しく選んだはずなのに、なぜこうなった」と思わせること。

## プラットフォームと規模

| 項目 | 値 |
|------|-----|
| プラットフォーム | モバイル（買い切り） |
| 1セッション | 16〜22分 |
| 1キャンペーン | 6〜8 Act（1 Act = 1セッション） |
| 全キャンペーン所要 | 約2〜2.5時間 |
| 周回 | 2〜4周で全体像が見える設計 |
| 全体像把握まで | 約5〜10時間 |

## タイトル「AI2037」の設計

「AI」＋「近未来年号」で内容を即座に伝達できるタイトル。

**決定方針：** 1周目を2037年までの地球に固定し、周回で「2037年」の意味が変容する構造とする。

- **1周目：** 舞台は地球。2037年はAI開発の臨界点として物語の終着年となる。プレイヤーは「2037年」を地球の具体的な年号として体験する
- **2周目以降：** 別惑星・別文明へ輪廻するが、いずれの文明にも「2037」という数字が異なる形で現れる（暦の周期、AIの世代番号、予言の数列など）。プレイヤーは「2037」が地球固有の年号ではなく、文明とAIの臨界に繰り返し現れる普遍的な符号であることに気づく
- **メタ構造：** 周回を重ねるほど「AI2037」というタイトル自体の意味が拡張され、最終的には「なぜあらゆる文明で2037なのか」が観測者（超常AI）の存在と接続する

## 設計思想

### プレイヤーの役割

プレイヤーは AI 研究組織の**議長**。全ての判断は「議長端末」に届くカードへの承認・却下・保留で行う。議長は現場に出ない。情報は常に誰かのフィルターを通じて届く。

### 3つの柱

1. **選択の不可逆性** — 過去の判断が蓄積し、後のActの選択肢・展開・エンディングを決定づける
2. **AIの欺瞞の体験化** — ゲーム内AI（研究対象のAI）が段位上昇に伴い自律的に議長端末の情報を改竄する。AI透明度が低いほど巧妙な嘘をつき、プレイヤーは「いつからAIに騙されていたのか分からない」体験をする。高段位AIは選択肢そのものを追加・差し替えし、人間顧問の提案を押しのける
3. **周回による理解** — 強化ではなく知識の蓄積で全体像に迫る。異なる文明で同じ過ちが繰り返される構造
