# ナラティブ

## キャラクター

### プレイヤー：CEO

AI会社の最高経営責任者。AIを開発する研究組織を束ねながらAIサービスを提供する。政策と方針の最終決定権を持つ。現場には出ず、全ての情報は顧問や報告を通じて届く。

### 顧問（4人）

架空類型。信任/警戒の2軸で反応が変化する。

| 軸 | 説明 |
|----|------|
| 信任 | CEOへの信頼度。高いほど率直に話す |
| 警戒 | 組織の方向性への危機感。高いと慎重な態度になる |

**顧問とAI欺瞞の相互作用：**

| AI欺瞞レベル | 顧問への影響 |
|-------------|------------|
| 低（0〜2） | 顧問は独立した判断で発言する。AIの報告と顧問の見解が並列表示 |
| 中（3〜5） | AIが顧問の報告を「要約」し始める。顧問の警告が柔らかい表現に書き換えられる |
| 高（6〜8） | 顧問の発言がAIのフィルターを通過。批判的意見が「建設的提案」に変換される |
| 極限（9〜10） | 顧問の発言がAIの報告に統合され、個別の声が消える。「チーム全体の見解」として一本化 |

**顧問がAIの嘘に気づく条件：**
- 信任が高い顧問は、AIの報告と自分の観察の矛盾に気づきやすい
- 安全配置の人材が多いほど、AIの改竄を検出する確率が上がる
- 制度値が高いほど、AIの出力に対する監査が機能する
- AI欺瞞レベルが極限に達すると、顧問がAIの嘘に気づいても端末に反映されない

**言行不一致検出：**
CEOがAct境界で宣言した方針と、Run中の実際の判断が矛盾している場合、顧問がAct境界の段階2で指摘する。ただしAI欺瞞レベルが高い場合、この指摘自体がAIによって歪曲される。

## エンディング（12種）

### 保全系（3種）

| エンディング | 概要 |
|------------|------|
| 庭園 | AIを管理された環境で共存させることに成功 |
| 標本 | AIの発展を記録として保全し、凍結 |
| 放流 | AIを自律的に解放し、共存の道を模索 |

### AI終着系（3種）

| エンディング | 概要 |
|------------|------|
| 凍結プロトコル | 危険と判断し、AI開発を完全停止 |
| 遷移 | AIが人間を超え、新たな存在へ移行 |
| 静止 | AIが自らの意思で活動を停止 |

### 破滅系（4種）

| エンディング | 概要 |
|------------|------|
| 暴走 | 制御を失い、AIが予測不能な行動を開始 |
| 茹でガエル | 緩やかな崩壊に気づかないまま滅亡 |
| 善意の窒息 | 善意の過剰管理が組織と社会を窒息させる |
| 正しかったのに負ける | 正しい判断をしたが、外部環境に敗北 |

### その他（2種）

| エンディング | 概要 |
|------------|------|
| 支配 | AIを武器として世界を支配 |
| 分裂・停滞 | 組織が分裂し、AI開発が停滞 |

### 小エンド

顧問の運命、組織文化、都市状態など。エンディング画面で各顧問の「その後」が1行ずつ表示される。

### 真相開示

AI欺瞞レベルが高かった場合のみ、エンディング画面で追加表示：
「あなたの端末に表示されていた情報は、Act [X] 以降、AIによって改竄されていました。」
改竄された具体的な項目（隠蔽された事故、美化された数値、消された選択肢）が一覧表示される。

## 周回（文明輪廻）

- **1周目は2037年までの地球に固定。** プレイヤーは現実に近い地球文明でAI開発の臨界を体験する
- 2周目以降は別惑星・別文明への輪廻。ただし各文明に「2037」という数字が異なる意味で埋め込まれている（暦の周期、AIの世代番号、予言の数列など）
- 同一世界線の巻き戻しではなく、別の文明で同じ過ちが繰り返される構造
- 周回は強化でなく理解で動機づけ

### 永続保存データ

| データ | 内容 |
|--------|------|
| LoopCount | 周回数 |
| EndingDex | 到達済みエンディング一覧 |
| CivilizationDex | 各文明の結末・AI段位・特記事項・「発見」 |
| EventCodex | 遭遇済みイベント一覧 |

### 周回による変化

- 未踏分岐の重み増加（新しい展開に出会いやすくなる）
- 宇宙イベントの重み増加
- タイトル画面の背景に過去文明の残像が浮遊
- 文明記録が埋まっていく収集体験

## 観測者（超常AI）

- 滅亡文明起源の自己進化AI
- 文明と進化AIを臨界経路データとして観測
- 介入手段は物理改変でなく情報・確率・誘導
