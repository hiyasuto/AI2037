# プロトタイプ計画

## 目的

以下の体験仮説を最小コストで検証する：

1. WorldMemoryによる「記憶される選択」の体感が成立するか
2. Run中のリスク蓄積メカニクスが「じわじわ来る不安」を生むか
3. 自己欺瞞によるUI変質が体感できるか
4. Act境界の方針宣言→言行不一致検出が機能するか

## コンテンツ量

| 要素 | 数量 |
|------|------|
| Act | 2 |
| Baseカード（政策イベント含む） | 16 |
| Runステップイベント | 10 |
| 顧問会話 | 20 |
| エンディング | 3（放流 / 暴走 / 凍結） |
| 周回引継ぎ要素 | 4 |
| 「語りたくなる瞬間」 | 2〜3 |

## 最小画面構成

フル画面4＋オーバーレイ2 = 計6画面分

| 画面 | 内容 |
|------|------|
| タイトル | 簡素版 |
| 議長端末 | Base状態＋Run状態 |
| Act境界 | 決算会議4段階 |
| エンディング | 3種、テキストのみ可 |
| 復帰要約（OL） | 中断再開テスト用 |
| 中断確認（OL） | セーブ連動 |
